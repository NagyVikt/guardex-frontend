# Guardex Frontend

Simple Next.js frontend for visualizing a guarded multi-agent workflow (execute, plan, merge, and installation flows) with a guided UI.

## Tech stack

- Next.js 15
- React 19
- TypeScript

## Quick start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Available scripts

- `npm run dev` - start local development server
- `npm run build` - create production build
- `npm run start` - start production server
- `npm run lint` - run ESLint
